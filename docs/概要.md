# プロジェクト概要：Preness (TOEFL ITP 学習プラットフォーム)

## 1. プロジェクト目的
本プロジェクトは、TOEFL ITPのスコアアップを目指す学習者のために、本番に近い形式での**模擬試験**と、弱点補強のための**セクション別演習**を提供するウェブアプリケーションです。

## 2. ターゲットユーザー
- TOEFL ITPを受験予定の学生・社会人
- 自身の現在の実力を正確に把握したい学習者
- 隙間時間に特定のパート（リスニング、文法、読解）を集中して解きたい学習者

## 3. 主要機能
- **模擬試験 (Mock)**: 3セクション（Listening, Structure, Reading）を連続して解く。
- **セクション別演習 (Exercises)**: 特定のパートのみを短時間で解く。
- **自動採点・解説**: 解答後すぐに正答率と間違えた問題の解説を表示（データの永続化は最小限）。
- **問題データ自動投入**: 外部システム（FastAPI）から最新の問題データをAPI経由で取得。

## 4. 設計方針
- **シンプルさの追求**: タイマー制限や複雑な音声再生制限を排除し、学習者が自身のペースで進められる体験を優先します。
- **データの蓄積と復元**: セクション完了ごとに回答をサーバーへ保存することで、不意のブラウザ終了時でも進行状況を保持し、再開時に復元できる体験を提供します。
- **セクション単位のサーバー保存**: 解答データはセクション遷移時にサーバーへ送信・永続化します。これにより、JavaScriptへの過度な依存を避け、堅牢なデータ管理を実現します。
