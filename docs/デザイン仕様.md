# アプリケーション共通デザイン仕様書 (Design System)

## 1. Design Concept
- **Style**: クリーン、モダン、フラットデザイン
- **Tone & Manner**: データを扱う分析・レポートアプリとして、ユーザーに信頼感と視認性の高さを提供する
- **Base Rule**: 余白を十分に活かし、情報を整理して表示する

## 2. Color Palette
Tailwind CSSの命名規則に準拠したベースカラー定義。

### Primary (主調色)
- `primary-base`: `#1E40AF` (主要ボタン、ヘッダー/フッター、重要なアイコン)
- `primary-light`: `#EFF6FF` (選択状態の背景、ハイライト)

### Accent / Status (アクセント・状態色)
- `accent-success`: `#10B981` (ポジティブな数値指標、完了ステータスなど)

### Neutral (背景・テキスト・枠線)
- `bg-default`: `#FFFFFF` (アプリ全体の基本背景色)
- `bg-muted`: `#F3F4F6` (セクション区切り、テーブルのヘッダー背景など)
- `text-main`: `#1F2937` (見出し、主要な本文)
- `text-sub`: `#6B7280` (補足テキスト、ラベル、プレースホルダー)
- `border-color`: `#E5E7EB` (カード、テーブル、フォームの境界線)

## 3. Typography
- **Font Family**: `Inter`, `Roboto`, `Noto Sans JP`, `sans-serif`

### Font Weights
- **Regular**: 400 (本文)
- **Medium**: 500 (ボタン、サブ見出し)
- **Bold**: 700 (主要な見出し)

### Font Sizes
- **h1 (ページタイトル)**: 24px (1.5rem), Bold
- **h2 (セクション見出し)**: 20px (1.25rem), Bold
- **h3 (カード見出し)**: 16px (1rem), Bold
- **body (基本テキスト)**: 14px (0.875rem), Regular
- **small (補足テキスト)**: 12px (0.75rem), Regular

## 4. UI Components

### 4.1. Buttons

#### Primary Button
- **Background Color**: `primary-base` (`#1E40AF`)
- **Text Color**: `#FFFFFF`
- **Border Radius**: 6px
- **Padding**: 上下 8px, 左右 16px
- **Font Weight**: Medium

#### Secondary/Outline Button
- **Background Color**: transparent
- **Border**: 1px solid + `border-color` (`#E5E7EB`)
- **Text Color**: `text-main` (`#1F2937`)
- **Border Radius**: 6px
- **Padding**: 上下 8px, 左右 16px
- **Font Weight**: Medium
