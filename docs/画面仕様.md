## 音声再生 共通UI仕様

| 状態 | 表示 |
|---|---|
| 未再生 | 🔊 `[▶ 再生]` ボタン（オレンジ） |
| 再生中 | 🔊 再生中...（スピナーアニメーション） |
| 再生済み | 🔊 再生済み（グレーアウト・無効化） |

- 一時停止・シークバーなし
- 再生中も選択肢は表示したまま（隠さない）

## セクション別演習 解答画面の構成

### Part A（短い会話 / 空所補充 / 正誤判定）

- 全問を1ページにまとめて表示
- 問題間はカード区切り
- 各問題カードに問題番号バッジ（`Q1`, `Q2` ...）
- 選択肢: 4択ラジオボタン（選択済みはオレンジハイライト）
- 上部に進捗バー（回答済み問題数 / 全問題数）
- 下部に「採点する」ボタン（全問回答で有効化）

### Part B・C（長い会話 / 講義）

- ページ上部にセット共通音声再生ボタン
- 各問題に個別音声がある場合は問題カード内に再生ボタン

### Reading（長文読解）

- 左カラム: パッセージ（スクロール固定・sticky）
- 右カラム: 問題カード縦並び
- カラム比率: パッセージ 45% / 問題 55%

## 模擬試験 タイマー表示

- ヘッダー固定表示（`⏱ 残り MM:SS`）
- 残り5分で文字色を赤に変更
- 残り1分でパルスアニメーション
