# 非機能要件

## 技術スタック

| 分類 | 技術 |
|---|---|
| バックエンド | Ruby on Rails 8.0.x |
| DB | PostgreSQL |
| 認証 | Devise |
| ファイル管理 | AWS S3（音声ファイル・直接参照） |
| フロントエンド | Tailwind CSS |
| 開発環境 | Docker |
| メール送信 | Resend（本番環境のみ） |

---

## 国際化・ローカライゼーション

- アプリの日本語化（i18n）
- 日時処理は全て日本時間（JST, UTC+9）
- タイムゾーン設定: `config.time_zone = "Tokyo"`

---

## アーキテクチャ方針

- サービス層を作成してコントローラーの肥大化を防ぐ（`app/services/` 配下）

---

## セキュリティ

| 項目 | 設定 |
|---|---|
| セッションタイムアウト | 24時間 |
| ログイン試行回数制限 | なし |
| 二段階認証 | なし |
| ソーシャルログイン | なし |
| APIアクセス制限 | 特定の送信元IPアドレスのみ許可（環境変数 `ALLOWED_CONTENT_SOURCE_IPS`） |

---

## ファイル管理

- 音声ファイルの実体は AWS S3 に保存。
- 本アプリは Active Storage を使用せず、DB に保存された `audio_url` を直接参照して再生を行う。
- パブリックアクセス（公開URL）または署名付きURLの運用とする。
- FastAPI 側で S3 へのアップロードを行い、本アプリへその URL を POST するフロー。

---

## 環境変数一覧

| 変数名 | 説明 |
|---|---|
| `DATABASE_URL` | PostgreSQL接続URL |
| `AWS_ACCESS_KEY_ID` | AWS S3アクセスキー |
| `AWS_SECRET_ACCESS_KEY` | AWS S3シークレットキー |
| `AWS_REGION` | AWS S3リージョン |
| `AWS_BUCKET` | AWS S3バケット名 |
| `RESEND_API_KEY` | Resend APIキー（本番環境のみ） |
| `ALLOWED_CONTENT_SOURCE_IPS` | 問題投入を許可するIPアドレス（カンマ区切り） |

---

## 信頼性・冪等性

- API等で重複リクエストが届いても安全に処理できるよう、適宜冪等性を保つ実装を行う。

---

## フロントエンドの注意事項

- 解答データはセクション遷移時または完了時にサーバー側へ送り、DBに永続化する。
- ページ再読み込み時はDBから解答データを取得して復元する。これによりJavaScriptへの依存を最小限に抑える。
- ターゲットデバイスはPCメイン（レスポンシブ対応はするが、PC体験を優先）
